Lista dos Melhores Sites que já vi

Top 3 dos melhores sites que existem na web

  1. The Useless Web
  2. Hacker Typer
  3. Shady URL

Principais motivos para eu gostar deles

  • The Useless Web
    Deste muita gente já ouviu falar. Ele é um site responsável por te levar até as páginas mais aleatórias e, digamos, “inúteis” da Internet. Desperdiçar tempo parece ser divertido de vez em quando, afinal.

  • Hacker Typer
    Este daqui é essencial para quem assiste a muitos filmes de hackers e, algum dia, já quis ser como eles. Agora ficou fácil realizar esse sonho! Para isso, basta acessar o site e digitar como se você soubesse exatamente o que está fazendo.

  • Shady URL
    Interessante para quem quer assustar a todos no grupo dos amigos. O Shady URL transforma qualquer endereço da web em um link suspeito, como se fosse vírus. Quer atenção e não sabe como ganhar? Basta mandar esse link para seus amigos e deixá-los preocupados achando que seu PC está com vírus.
